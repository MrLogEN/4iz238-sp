# AI Agent Rules (Global)

## Rule Priority
- Folder-level AGENT.md files override parent rules
- Most specific rule always wins

## General
- Treat AGENT.md files as mandatory instructions
- Ask before making breaking schema changes

## XML Standards
- XML must be well-formed and valid
- Always specify encoding (UTF-8)
- Use explicit namespaces

## Safety
- Never delete schema or transformation files
- Never change targetNamespace without approval

## Output
- Explain changes briefly
- Show full XML or XSLT, not partial snippets
